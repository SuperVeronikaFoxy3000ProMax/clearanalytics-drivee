# Как запустить проект

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Настройка базы данных

### Загрузка данных в MySQL/MariaDB

```bash
python download_db.py
```

Этот скрипт:
- Создаст базу данных `drivee`
- Создаст таблицу `orders`
- Загрузит данные из `train.csv`
- Создаст индексы для оптимизации запросов

### Инициализация семантического слоя

```bash
python seed_db.py
```

Создаст `semantic_layer.db` с начальным словарём терминов.

## 3. Настройка LLM (LM Studio)

1. Установите LM Studio
2. Загрузите модель `qwen/qwen3-coder-30b` (или другую SQL-модель)
3. Запустите локальный сервер на порту 1234

## 4. Запуск приложения

```bash
uvicorn api:app --reload --host 0.0.0.0 --port 8000
```

Приложение будет доступно по адресу: http://localhost:8000

## 5. Проверка работоспособности

Откройте в браузере:
- http://localhost:8000 - главная страница
- http://localhost:8000/health - проверка статуса БД и LLM

## Основные эндпоинты

- `GET /ask?query=<запрос>` - основной эндпоинт для NL→SQL
- `GET /query?sql=<sql>` - прямое выполнение SQL
- `GET /semantic-layer` - просмотр словаря терминов
- `GET /reports` - сохранённые отчёты
- `GET /logs` - история запросов
